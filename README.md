智能春节祝酒词生成器 — 功能说明文档

一款纯前端、离线可用、节日氛围浓厚的祝酒词智能生成工具，专为春节聚会场景设计。

🎯 核心功能

用户输入以下三项信息：

- 称呼（如：王叔叔、李老师）
- 年龄（可选，用于判断年龄段）
- 职业/身份（如：退休医生、程序员、创业者）

系统将从 1000+ 条精心编写的春节祝酒词语料库 中，智能匹配最贴合的一条，并自动将主语统一替换为用户提供的称呼，生成一段自然、得体、喜庆的祝酒词。

✨ 特色亮点

1. 智能语义匹配（非模板替换）
- 祝酒词按 职业标签（如“老师”“医生”“程序员”）、年龄段（青年/中年/长辈）和 节日主题（春节）进行分类；
- 匹配优先级：职业 > 年龄段 > 通用；
- 从匹配池中随机抽取一条，避免重复感。

2. 超大语料库（1050+ 条）
- 200+ 条高质量手工编写样本，覆盖：
  - 职业：教师、医生、程序员、老板、农民、警察、军人、厨师、设计师、金融从业者等；
  - 身份：退休人员、学生、亲友、长辈等；
  - 场景：家庭聚会、职场团拜、朋友聚餐等。
- 850+ 条程序生成变体，基于祝福语组合，确保语言丰富、不机械。

3. 主语统一处理
- 所有祝酒词使用 {{NAME}} 作为唯一占位符；
- 生成时全局替换为用户输入的称呼；
- 彻底移除“您”“他/她”“咱们的XX”等冗余指代，确保语句聚焦、自然。

4. 沉浸式春节视觉体验
- 页面左右两侧显示动态春联（5 组候选，每次刷新随机选用一组）；
- 顶部点缀红灯笼 emoji；
- 红金配色 + 渐变背景，营造浓厚年味；
- 核心功能区域垂直+水平居中，布局清爽。

5. 完全离线 & 隐私安全
- 无需网络请求，所有数据内嵌于 HTML 文件；
- 无 API 调用，不收集用户信息；
- 单文件部署，双击即可运行。

📱 使用方式

1. 将完整代码保存为 spring-toast-generator.html；
2. 双击用浏览器打开；
3. 填写称呼、年龄、职业；
4. 点击「生成祝酒词」按钮，即可获得专属祝福语。

💡 提示：职业字段支持模糊匹配（如输入“退休教师”会同时触发“退休”和“老师”标签）。

输入

- 年龄：62
- 职业：退休教师
